# Work Order Consumed Materials with Scrap

This report shows all materials consumed in Work Orders, including:
- Required items (from Work Order Items table)
- Consumed items (from Stock Entry)
- Scrap items (from Work Order Scrap Items table)
- Department information

## Filters
- Company (required)
- From Date (required)
- To Date (required)
- Work Order (optional)
- Department (optional)

## Columns
- Work Order
- Department
- Production Item
- Item Code
- Item Name
- Type (Required/Consumed/Scrap)
- Required Qty
- Consumed Qty
- Variance
- UOM
